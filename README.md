 Bangalore Colleges Map 

An interactive map showing colleges in Bangalore with search functionality.

Features
- Interactive map with college locations
- Search bar to find colleges by name
- Click markers for college details
- Responsive design works on all devices


Technologies Used
- HTML5, CSS3, JavaScript
- Leaflet.js for mapping
- Leaflet-Search for search functionality
- OpenStreetMap for map tiles

How to Use
1. Open the map
2. Use search box to find colleges
3. Click on markers for details
4. Zoom in/out to explore

 Data Source
College data is stored in `colleges.json` with:
- College Name
- Location
- Latitude & Longitude
